# Scan & Validation Service

Service de scan et validation pour Event Planner - Validation des tickets et contrôle d'accès.

## Fonctionnalités

- Scan QR codes
- Validation tickets
- Contrôle d'accès
- Mode offline
- Synchronisation
- Anti-fraude

## Stack Technique

- Node.js + Express
- Camera API
- QR Code Scanner
- PostgreSQL
- Redis (cache)

## Installation

```bash
npm install
npm start
```

## Développement

```bash
npm run dev
```

## Docker

```bash
docker-compose up -d
```
